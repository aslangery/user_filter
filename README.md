# user_filter
Фильтры реализуются в классе **Filter**. Методы класса собирают условие WHERE для запроса к временной таблице. 
В методы передаются 3 параметра: _ключ_, _значение_ и _условие связки_.
Список фильтров соответствует полям временной таблицы и легко может
быть расширен, путем изменения запроса на формирование временной таблицы.
Поиск осуществляет метод search() класса **Search**. В метод передается 
массив столбцов для выборки и сформированный объект **Filter**. Метод возвращает объекты результаты запроса
или _null_ случае ошибки.
